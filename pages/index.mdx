# Atomize Icons

Meet [atomize icons](https://github.com/nzaih1999/atomize_icons) The word most complete set of Web3 icons. Designed for the community, by the community.

## Installation?

To install all you need to do is run the following command in your terminal of choice:

    <Tabs items={["pnpm", "npm", "yarn"]} >
      <Tab >
      ```bash copy 
        pnpm add atomize_icons

      ```
      </Tab>
      <Tab>

      ```bash copy
        npm install atomize_icons

      ```
      </Tab>
      <Tab>
       ```bash copy
        yarn add atomize_icons

      ```
      </Tab>
    </Tabs>


    ## Usage


```jsx copy
import { About } from "atomize_icons";

function App() {
  return (
    <h1>
      What are you talking <About /> ?
    </h1>
  );
}
```
